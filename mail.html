Checking CSE Webmail between 12am and 6am

This website is mainly for my usage, so I shall dive right in.

  1. SSH vyom.cc.iitk.ac.in
  2. From vyom, SSH turing.cse.iitk.ac.in
  3. Use the following command:
    $ mutt -f imap://imap.cse.iitk.ac.in
  4. You're done!
  5. For checking IITK Webmail (because using Terminal is fun), you can use:
    $ mutt -f imap://newmailhost.cc.iitk.ac.in
